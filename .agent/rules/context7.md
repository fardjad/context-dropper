# Context7 Rule

These rules define when and how the Context7 MCP server must be used for library documentation and code generation.

## 1. Required Usage

- The Context7 MCP server must be used to retrieve documentation, API references, or code generation examples for external libraries and tools.
- Setup or configuration steps for libraries and APIs must use Context7 to verify correct implementation.
- The use of Context7 for documentation and problem-solving must happen automatically without explicit instruction from the user.
